# LumaNest

LumaNest is a warm, AI-powered journaling and wellness platform built with Flask, SQLite, and Hugging Face Transformers.

## What this version includes

- email and phone based authentication
- private per-user data storage
- multilingual text and voice journaling input
- AI analysis for emotion, sentiment, and life themes
- weekly mood tracker summaries
- weekly sleep tracker summaries
- habit points and partner-style reward redemption
- soothing music tab with an in-app player

## Run locally

1. Activate the virtual environment:
   `.\venv\Scripts\Activate.ps1`
2. Install packages:
   `pip install -r requirements.txt`
3. Start the app:
   `python app.py`
4. Open:
   `http://127.0.0.1:5000`
